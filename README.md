# 最直观的 Typecho 主题开发框架  

> The Intuitive Typecho Theme Framework.  

✨ **核心特性**  
- 🧩 组件化
- 🛣️ 自动路由
- 🔧 配置管理
- 🔌 内置API支持

**运行环境**  
PHP 8.1+ / Typecho 1.2+

当前为开发版本，生产环境请使用[稳定版](https://github.com/YuiNijika/TTDF/releases)
